- 顶层 package.json 仅声明前端依赖与 dev/build/start 脚本，实际构建由 Next.js 驱动；后端位于独立子模块，默认监听 3333 端口。
- next.config.mjs 中的 rewrites 把浏览器请求 /api/:path* 和 /uploads/:path* 反向代理到 http://localhost:3333，使前后端在开发时以单入口 3000 端口对外暴露，形成「前端直出 + 后端 API」的解耦架构。
- test-results/ 与 ms-playwright/ 为 Playwright 自动化测试产物与 Chromium 二进制缓存，覆盖文章撰写流程与后端 API 场景，错误上下文统一归档于 error-context.md。
- docs/ 存放架构、数据模型与重构方案等 Markdown 文档，作为跨前后端的单一事实来源。
