- API 调用统一通过 `api/client.js` 导出的命名空间对象（如 `postsApi.list`、`auth.login`），禁止在页面中直接写 fetch 或拼接 URL。
- 每个 API 方法内部使用 `URLSearchParams` 组装查询参数，空参数时省略 `?`，保持 URL 整洁。
- 组件目录采用 `components/<PascalCase>/<Component>.jsx` + 同名 `<Component>.module.css` 的自包含结构，样式通过 `import styles from './xxx.module.css'` 使用。
- 需要访问浏览器 API 的组件顶部声明 `'use client'`，并在 `AuthProvider` 等 Provider 外层包裹于 `app/providers.jsx` 中集中注册。
- 错误处理统一抛出 `ApiError`（携带 status/code/message），调用方通过 try/catch 捕获并根据 code 分支处理。
