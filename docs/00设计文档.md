# 墨客博客 — 设计文档

---

## 一、产品概述

### 1.1 产品定位

墨客博客是一款面向中文技术写作者的轻量级全栈博客系统，旨在提供极简的写作体验、美观的阅读界面和完整的内容管理能力。

### 1.2 核心价值

| 价值维度 | 描述 |
|---------|------|
| **极简写作** | 前端图片压缩、流畅的文章发布流程、Markdown 编辑器支持 |
| **内容自主可控** | 用户数据自托管 PostgreSQL，源码级可定制，无平台绑定 |
| **美观易用** | 基于 8 点网格系统的清新 UI 设计，支持响应式布局 |
| **多用户支持** | 每位用户拥有独立的个人页面 `/u/:username/` |
| **管理员功能** | 完整的管理面板 `/admin`，支持文章、评论、分类管理 |

### 1.3 用户角色

| 角色 | 描述 | 核心能力 |
|------|------|---------|
| **普通用户** | 浏览者、评论者、写作者 | 浏览文章、发表评论、发布文章、管理个人资料 |
| **管理员** | 系统管理员 | 管理所有文章、评论、分类，查看统计数据 |

## 二、设计原则

### 2.1 UI 设计原则

- **视觉层次清晰**：通过字体大小、颜色对比、间距建立清晰的信息层级
- **一致性**：按钮样式、颜色方案、交互模式保持统一
- **响应式**：适配桌面端、平板、手机等多种设备
- **留白**：适当的留白提升可读性和视觉舒适度
- **微动效**：导航栏滚动效果、Toast 提示、按钮悬停等增强交互体验

### 2.2 交互设计原则

- **即时反馈**：操作后立即给出视觉或文字反馈
- **乐观更新**：评论等操作先更新 UI，再异步提交
- **渐进增强**：核心功能优先，高级功能逐步扩展
- **错误恢复**：友好的错误提示，支持重试操作

### 2.3 色彩方案

| 色彩 | 用途 |
|------|------|
| 主色 `#2563eb` | 按钮、链接、强调元素 |
| 成功色 `#10b981` | 成功提示、状态标识 |
| 警告色 `#f59e0b` | 警告提示 |
| 错误色 `#ef4444` | 错误提示、删除操作 |
| 深色主题 `#1e293b` | 暗色模式背景 |

## 四、页面路由设计

### 4.1 公共页面

| 路由 | 页面 | 说明 |
|------|------|------|
| `/` | 首页 | 文章列表 |
| `/page/:pageNum` | 首页分页 | 文章列表分页 |
| `/category/:slug` | 分类页面 | 按分类浏览文章 |
| `/category/:slug/page/:pageNum` | 分类分页 | 分类文章分页 |
| `/post/:slug` | 文章详情 | 文章阅读 |
| `/about` | 关于页面 | 站点介绍 |

### 4.2 用户个人页面

| 路由 | 页面 | 说明 |
|------|------|------|
| `/u/:username` | 用户个人页面 | 展示用户信息和文章 |
| `/u/:username/posts` | 用户文章列表 | 用户文章分页 |
| `/u/:username/profile` | 个人资料管理 | 编辑个人信息、管理文章 |
| `/u/:username/write` | 写文章 | 创建新文章 |
| `/u/:username/write/:id` | 编辑文章 | 修改已有文章 |

### 4.3 管理页面

| 路由 | 页面 | 说明 |
|------|------|------|
| `/admin` | 管理面板 | 数据统计、文章/评论/分类管理 |
